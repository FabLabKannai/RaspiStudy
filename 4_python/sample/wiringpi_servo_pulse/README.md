# servo pulse - RasPi Python sample

Test for Continuous Rotation Servo <br/>
using WiringPi on Rapberry Pi <br/>
Command specify pulse <br>

### Usage
format : digits <br>
example <br>
- 200 : clockwide full speed <br>
- 300 : stop <br>
- 400 : anticlockwide low speed <br>
